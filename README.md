# Quiz Ariana Meble
Android quiz application with furnitures names to guess. Furnitures placed in quiz come from polish company named 'Ariana Meble'. 

## Technologies
* Android
* SQLite

## Synopsis 
* 10 questions 
* implicitly 4 answers
* possibility to choose categories and number of answers
